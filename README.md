# 🧠 Learn-AI-with-Chun

### *From Raw Data to Neural Networks: A Machine Learning Logbook*

Welcome to my digital laboratory. This repository is more than just homework; it is a chronicle of experiments, ranging from building neurons from scratch to analyzing complex financial datasets, data visualization, and data mining techniques. Here, you will find the code behind the concepts.

-----

## 🗺️ The Atlas (Folder Structure)

### 1\. 🎓 The Bootcamp: `DataScienceProgramming/`
*Where the foundations are laid. This directory tracks the evolution from basic Python scripting to full data science pipelines.*
  * **The Weekly Grind:** `week_2` through `week_13` (Source code for progressive lab exercises).
  * **Data Storytelling:**
      * `financial_data_eda.ipynb`: 📊 **Must Read.** A deep dive into financial datasets using Exploratory Data Analysis.
      * `scores_hist.png` & `student_scores.csv`: Visualizing academic performance distributions.
  * **Automation & Utilities:**
      * `canvasAPI.py`: A custom script to interact with the Canvas LMS API.
      * `training.py`: The engine room for lab training routines.

### 2\. 🎨 Visualizing Insights: `Data Viz/`
*Transforming raw data into clear, compelling stories.*
  * **COVID & Climate Data Analysis:** Interactive notebooks covering COVID-19 trends (`Week2_lab_assignment_...ipynb`), CO2 emissions, and global land temperatures.
  * **Datasets:** `owid-covid-data.csv`, `owid-co2-data.csv`, and `GlobalLandTemperaturesByCountry.csv`.

### 3\. ⛏️ Digging for Patterns: `Data Mining/`
*Extracting meaningful information from unstructured and structured datasets.*
  * **Text Processing Labs:** E.g., `COMP4040_Lab2_WordCount.ipynb` demonstrating fundamental frequency analysis and NLP basics.

### 4\. 🚀 The Proving Ground: `Project/`
*Real-world applications and domain-specific prototypes.*
  * **🏦 `Banking/`**: Experiments in Fintech. Credit scoring, fraud detection, or customer segmentation models live here.
  * **💬 `Zalo_AI/`**: Prototypes focusing on the Zalo platform—likely involving NLP or user behavior modeling.
  * **🏥 `BVK/` & `TOMPEI-CMMD/`**: Specialized healthcare/medical data projects.
  * **🛠️ `fusedModel.py`**: A hybrid model architecture combining multiple approaches for better inference.
  * **🧪 `Stuff/` & `draft/`**: The scratchpad. Raw ideas, failed experiments, and "what if" code.

### 5\. ⚡ The Core: Root Directory
*Lightweight tools and standalone projects.*
  * `neuron.py` & `Numerical.py`: **Back to Basics.** Mathematical implementations of neural units and numerical methods without heavy framework abstraction.
  * `project2-notebook.ipynb`: A comprehensive standalone project (The "Capstone").
