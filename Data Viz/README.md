# Data Visualization

This folder contains assignments, projects, and lab work related to Data Visualization, organized by week and topic. Each subfolder contains Jupyter Notebooks encompassing the analysis and the corresponding datasets used.

## Folder Structure

### `Homework1/`
Contains the first homework assignment, focusing on data cleaning, processing, and visualization tasks.
*   **`Homework1_V202401566.ipynb`**: The main notebook containing the solutions for Homework 1. Highlights include data cleaning (converting prices, destructuring designer columns) and creating impactful visualizations like horizontal lollipop charts, box plots, and a scatter plot analyzing the correlation between item volume and price.
*   **`ikea_data.csv`**: The dataset used for Homework 1, containing information about various IKEA furniture products, prices, designers, and metrics.

### `week1/`
Contains the lab assignment and datasets for the first week.
*   **`Week1_lab_assignment_V202401566_V202401461.ipynb`**: The Jupyter Notebook containing the lab work and exercises for Week 1.
*   **`GlobalLandTemperaturesByCountry.csv`**: Dataset containing historical global land temperature data by country.
*   **`owid-co2-data.csv`**: Dataset from Our World in Data (OWID) containing comprehensive historical statistics on CO2 emissions.

### `week2/`
Contains the lab assignment and datasets for the second week.
*   **`Week2_lab_assignment_V202401566_V202401461.ipynb`**: The Jupyter Notebook containing the lab work and exercises for Week 2. Based on previous chat context, this includes tasks like visualizing yearly new COVID-19 cases using custom-formatted charts.
*   **`owid-covid-data.csv`**: Dataset from Our World in Data (OWID) containing detailed global data on the COVID-19 pandemic.
